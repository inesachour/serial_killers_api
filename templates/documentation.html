API Documentation

Access and query the Serial Killers Database

Get All Subjects

Retrieves a complete list of all serial killers in the database.

GET /api/killers/

Response Example

[
    {
        "id": 1,
        "common_name": "Ted Bundy",
        "full_name": "Theodore Robert Bundy",
        "aliases": "The Lady Killer, The Campus Killer",
        "country": "USA",
        "proven_victims": 30,
        "possible_victims": "100+",
        "status": "Deceased"
    },
    ...
]

Search Subject by Name

Searches the database for a specific serial killer by name (supports partial matches).

GET /api/killers/<common_name>/

Response Example

[
    {
        "id": 1,
        "common_name": "Ted Bundy",
        "full_name": "Theodore Robert Bundy",
        "aliases": "The Lady Killer, The Campus Killer",
        "country": "USA",
        "proven_victims": 30,
        "possible_victims": "100+",
        "status": "Deceased"
    }
]

Database Response:

Data Fields Reference

Description of fields returned in the JSON response objects.

Field Type Description Notes
common_name string Commonly known name of the subject e.g. "Ted Bundy"
full_name string Full legal name of the subject
aliases string Known aliases or nicknames Comma-separated string
birth_country string Country of birth e.g., "USA", "United Kingdom"
gender string Gender of the subject Values: "male", "female", "unknown"
proven_victims integer Confirmed number of victims Exact count only
possible_victims string Estimated victim count ⚠️ Can be "17", "100+", "50-60"
years_active string Period of criminal activity ⚠️ "1990-2001" (continuous) or "1990,2001" (separate years)
active_countries string Countries where crimes occurred Comma-separated: "USA, Canada"
modus_operandi string Method of operation e.g., "Strangulation", "Poisoning"
capture_date date Date of apprehension YYYY-MM-DD format
sentence string Legal sentence received e.g., "Life imprisonment", "Death penalty"
status string Current status Values: "Imprisoned", "Deceased", "At Large", "Released", "Unknown"
death_date date Date of death YYYY-MM-DD format, only if "Deceased"
manner_of_death string How the subject died e.g., "Executed", "Natural causes", "Suicide"
notes string Additional details or context Free-form text field
